# Generated Artifacts

This directory is for local run outputs:

- curriculum manifests
- expert evaluation reports
- collected datasets
- trained checkpoints
- exported runtime artifacts

These files are useful for iteration, but they are not the source of truth for
the publishable training pipeline. Many records in this directory are
machine-specific or point to historical local paths, so this directory is
ignored by default through [`../.gitignore`](../.gitignore).
